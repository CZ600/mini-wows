# 教程页面设计文档

## 概述

在游戏首页新增教程按钮，点击后跳转到独立的教程页面，展示游戏的详细操作说明。

## 布局设计

采用左侧目录 + 右侧滚动内容的布局：

```
┌─────────────────────────────────────────────────────────┐
│  ← 返回首页                              3D 海战 - 教程  │
├──────────────┬──────────────────────────────────────────│
│              │                                          │
│   目录大纲    │   教程内容（长页面，滚动查看）               │
│              │                                          │
│  • 移动控制   │   【移动控制】                             │
│  • 开火操作   │   W - 加速   S - 减速                     │
│  • 瞄准系统   │   A - 左转   D - 右转                     │
│  • 鱼雷发射   │   ...                                    │
│  • 技能使用   │                                          │
│  • 单人模式   │   【开火操作】                             │
│  • 多人模式   │   鼠标左键 - 开火                         │
│  • 联机流程   │   ...                                    │
│              │                                          │
└──────────────┴──────────────────────────────────────────┘
```

## 交互设计

1. 点击左侧目录项，右侧平滑滚动到对应章节
2. 右侧内容区域可自由滚动
3. 滚动时，左侧目录自动高亮当前可见章节
4. 顶部有返回首页按钮

## 内容组织

共 8 个章节：

### 1. 移动控制
- W：加速（提升档位）
- S：减速（降低档位）
- A：左转
- D：右转
- 档位系统说明（0-5档）

### 2. 开火操作
- 鼠标左键：发射主炮
- 主炮自动瞄准准星方向
- 弹道受距离和风速影响

### 3. 瞄准系统
- 鼠标右键：开启/关闭瞄准镜
- 瞄准镜下可缩放视野（鼠标滚轮）
- Q/E：调整瞄准镜高度
- 瞄准镜刻度线说明

### 4. 鱼雷发射
- 数字键 2/3/4：选择鱼雷等级
- 再次按下切换散射角度（窄/宽）
- 鱼雷特性说明（速度/射程/伤害）

### 5. 技能使用
- F：急速射击技能
- G：损害管制技能
- H：精准射击技能
- 技能冷却和使用时机

### 6. 单人模式
- 关卡系统（1-10关）
- 敌人类型和行为
- 升级机制
- 舰船类型解锁（4级后可选）

### 7. 多人模式
- 舰船类型：驱逐舰/巡洋舰/战列舰
- 各类型特点和定位
- 团队配合要点

### 8. 联机流程
- 快速匹配
- 创建房间
- 加入房间
- 房间内操作（准备、开始等）

## 路由配置

```jsx
<Route path="/tutorial" element={<TutorialPage />} />
```

不需要认证保护，未登录用户也可查看。

## 样式设计

### 整体布局
- 左侧目录宽度：250px，固定定位
- 右侧内容区域：自适应宽度，可滚动
- 背景色：深色主题，与游戏风格一致

### 目录样式
- 悬停效果：背景色变亮
- 当前章节：左侧边框高亮 + 背景色变化
- 字体大小：14px

### 内容样式
- 章节标题：24px，加粗
- 副标题：18px
- 正文：16px，行高 1.6
- 按键图示：内联显示，与文字对齐

## 技术实现

### 滚动监听

使用 `IntersectionObserver` 监听各章节的可见性，自动更新左侧目录的高亮状态。

### 平滑滚动

使用 `scrollIntoView({ behavior: 'smooth' })` 实现点击目录后的平滑滚动。

### 响应式设计

暂不考虑移动端适配，优先保证桌面端体验。
